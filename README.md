# ProdsGPT

**ProdsGPT** is a web-based tool that connects creators’ music and audio files directly to TikTok through the **TikTok Content Posting API**.

It allows users to:
- 🎵 Upload MP3 or WAV files through a simple web interface.  
- 🚀 Automatically publish audio/video posts to TikTok using TikTok’s official API.  
- 🔐 Authenticate securely via TikTok Login Kit.  
- ⚙️ Post content automatically on behalf of the user (with permission).

---

## 🔧 Technical Setup
- **Frontend:** HTML/CSS (static GitHub Pages site)  
- **Backend/API:** TikTok Developer APIs (Login Kit + Content Posting API)  
- **Hosting:** GitHub Pages (`https://prodofthecentury.github.io/prods-gpt-site/`)

---

## 🔒 Privacy & Security
ProdsGPT does **not** collect or store any personal information.  
All data is handled directly between the user and TikTok’s API endpoints.  
See the [Privacy Policy](https://prodofthecentury.github.io/prods-gpt-site/privacy.html) and [Terms of Service](https://prodofthecentury.github.io/prods-gpt-site/terms.html) for details.
